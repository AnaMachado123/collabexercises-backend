CollabExercises — Backend

Este repositório contém o backend da plataforma CollabExercises, uma aplicação destinada à gestão colaborativa de exercícios académicos. O backend foi desenvolvido para fornecer uma API segura e escalável que suporta as funcionalidades principais da plataforma, incluindo a colaboração entre estudantes, organização de conteúdos e autenticação segura. 

A API é responsável pela autenticação, gestão de utilizadores, armazenamento de exercícios, soluções e comunicação com a base de dados.

Tecnologias Utilizadas

Node.js

Express

MongoDB com Mongoose

JSON Web Tokens (JWT)

CORS

Nodemon (ambiente de desenvolvimento)

Instalação e Execução
1. Instalar dependências
npm install

2. Executar em ambiente de desenvolvimento
npm run dev

3. Executar em ambiente de produção
npm start
